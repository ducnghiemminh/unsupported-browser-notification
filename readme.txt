This plugin allow your website display a browser update notification when the browser of customer does not support for your website.

== Description ==

Unsupported Browser Notification is the best solution for the modern website which does not compatible with the old browsers of user.

* Easy to use
* Display notification when the browser of user does not support for your website
* Allow you to select the browser versions which support for your website
* Easily customize the notification message and display options

== Installation ==

1. Upload `oe-supported-browser` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I update the browser versions supported for my website? =

You can update the browser versions in the "Unsupported Browser Notification" setting page which is accessed by clicking on the "Unsupported Browser Notification" menu. 

= How can I customize the notification message? =

The default message is set by default when the plugin activate. You can update it in "Custom Setting" section in the setting page.

= Can I add a new different browser? =

The browsers is set by default so there's no way you can do it. But our team will work hard to update the browser number in some next versions.
